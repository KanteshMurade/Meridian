# 🚀 Meridian.ai - AI-Powered Code Review and Bug Suggestion System

> Intelligent AI-driven platform for automated code review, bug detection, refactoring suggestions, and severity analysis.

---

## 📌 Overview

 **Meridian.ai** is a smart web-based platform designed to help developers automate the code review process using Artificial Intelligence and Large Language Models (LLMs).

Instead of spending hours manually reviewing code, developers can upload or paste their code and instantly receive:

- 🐞 Bug Detection
- ⚠️ Severity Analysis
- ✨ Refactoring Suggestions
- 📊 Code Quality Insights
- 🔍 Side-by-Side Diff Comparison
- 📁 Review History & Team Sharing

This project improves development efficiency, reduces debugging effort, and promotes clean coding practices.

# 🏗️ System Architecture

The project follows a modular architecture with separate services for frontend, backend, and AI analysis.

```text
User
   │
   ▼
Frontend (React + Vite)
   │
   ▼
Backend API (Node.js + Express)
   │
   ▼
AI Service (Python + LLM)
   │
   ▼
Database / Storage
```

---

# 🔄 Workflow

1. User uploads or pastes code
2. System detects programming language
3. Backend validates and preprocesses code
4. AI engine analyzes source code
5. Bugs and bad practices are detected
6. Severity score is assigned
7. Suggestions and fixes are generated
8. Diff viewer compares original vs improved code
9. Results are displayed to the user
10. Review history can be saved/shared

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Basit07861/Meridian.git
cd Meridian
```

---

## 2️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## 3️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

Backend runs on:

```bash
http://localhost:5000
```

---

## 4️⃣ Setup AI Service

```bash
cd ai-service
pip install -r requirements.txt
python app.py
```

AI service runs on:

```bash
http://localhost:8000
```
